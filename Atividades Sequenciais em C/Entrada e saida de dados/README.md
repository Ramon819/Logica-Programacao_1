# Entrada e Saída de dados

Esta pasta reúne **exercícios em C** que desenvolvi quando comecei meus estudos em 2025.
Encontrei esses códigos antigos e decidi postá-los para comparar meu progresso ao longo do aprendizado.

Os programas abordam **operações matemáticas, médias, áreas de figuras geométricas, frações, juros e manipulação de tempo**.

## 🚀 Como executar

1. Faça o download do repositório ou clone:

```bash
git clone <URL_DO_REPOSITORIO>
```

2. Compile o arquivo desejado (exemplo usando GCC):

```bash
gcc produto2num.c -o produto2num
```

3. Execute o programa:

```bash
./produto2num
```

Repita o processo para cada arquivo que quiser testar.
